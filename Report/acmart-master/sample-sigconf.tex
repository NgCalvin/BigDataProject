\begin{abstract}
This project uitilize the Amazon Fine Food data set from Kaggle to predict rating based one user's comment. We will be using varies algorithm from big data to anaylsis the data set. In the end of this project, we are expected to observe the relationship between user's comments and rating. By comparing the result to test data we can do prediction of rating based on the similarity of the comments. Moreover, we will come up with the most and least popularity of products based on rating.
\end{abstract}
